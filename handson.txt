1. configurar o google Drive para o nosso DVC

    1. Criar pasta chamada remote-storage-dvc
    2. Copiar o URL depois de /folder/ ---> 19qSmBWtjo3Y_WgmG0uN8Q-wyLATmLEqh
    3. No terminal do DVC:

        rm -rf .dvc                                                                                                                                                                                           ─╯
        rm -f .dvcignore
        dvc init --subdir
        dvc remote remove storage    
        dvc remote add -d storage gdrive://19qSmBWtjo3Y_WgmG0uN8Q-wyLATmLEqh

        Checar a lista de storages configuradas
        dvc remote list

        checando o arquivo de config após a alteração
        ls -la
        cd .dvc
        cat config
