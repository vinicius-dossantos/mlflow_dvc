1. configurar o google Drive para o nosso DVC

    1. Criar pasta chamada remote-storage-dvc
    2. Copiar o URL depois de /folder/ ---> 19qSmBWtjo3Y_WgmG0uN8Q-wyLATmLEqh
    3. No terminal do DVC:

        cd ~/Documents/GitHub/mlflow_dvc

        rm -rf .dvc
        rm -f .dvcignore
        dvc init

        dvc remote remove storage    
        dvc remote add -d storage gdrive://19qSmBWtjo3Y_WgmG0uN8Q-wyLATmLEqh

        Checar a lista de storages configuradas
        dvc remote list

        checando o arquivo de config após a alteração
        ls -la
        cd .dvc
        cat config

        Já temos um csv de clientes data/external/clientes.csv 

        dvc add data/external/clientes.csv  

        se der erro: 
            git check-ignore -v data/external/clientes.csv.dvc

        O arquivo csv foi adicionado ao controle de versão do DVC, agr precisamos adicionar ao git o arquivo de metadados, que faz esse controle do DVC

        git add data/external/clientes.csv.dvc
        
        removendo o csv do controle do git caso de erro: 
            git rm --cached data/external/clientes.csv
